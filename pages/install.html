Downloading

The latest stable version of the source code for MATLAB/Octave is available in [compressed format here](https://git.io/v2Nym).

If you have git installed and would rather have the latest stable _FECGSYN_, you may consider cloning branch master from GitHub. ```bash git clone https://github.com/fernandoandreotti/fecgsyn.git ```

Installing

Install on Matlab

Simply extract the compressed package or include your git folder to Matlab/Octave's path. Matlab users can download and install the toolbox using the following lines of code:

{% highlight matlab %} cd('') fecgsyn_url='https://git.io/v2Nym'; urlwrite(fecgsyn_url,'fecgsyn_toolbox.zip'); unzip('fecgsyn_toolbox.zip'); cd fecgsyn_toolbox addpath(pwd);savepath {% endhighlight %}

Downloading

The latest stable version of the source code for MATLAB/Octave is available in compressed format.

If you have git installed and would rather have the latest stable FECGSYN, you may consider cloning branch master from GitHub:

{% highlight bash %} git clone https://github.com/fernandoandreotti/fecgsyn.git {% endhighlight %}

Installing

Install on Matlab

Simply extract the compressed package or include your git folder to Matlab/Octave's path. Matlab users can download and install the toolbox using the following lines of code:

{% highlight matlab %} cd('') fecgsyn_url='https://git.io/v2Nym'; urlwrite(fecgsyn_url,'fecgsyn_toolbox.zip'); unzip('fecgsyn_toolbox.zip'); cd fecgsyn_toolbox addpath(pwd);savepath {% endhighlight %} After installing FECGSYN you may verify your installation by running: {% highlight matlab %} fecgsyntest % tests FECGSYN {% endhighlight %}
